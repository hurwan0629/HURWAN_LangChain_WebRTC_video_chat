> 이제는 그룹 통화방을 구현하겠습니다.

```js
rooms = Map<roomCode, {
  roomId: int,
  roomCode: string,
  router: mediasoup.worker.Router | null
  passwordHash: string,
  status: "waiting" | "active" | "closed",
  hostUserId: users.id,
  peers: Map<userId, {
	  socketId: Set<Socket.id>,
	  nickname: 사용자 입력,
	  sendTransport: producer에 쓰이는 transport | null,
	  recvTransport: consumer에 쓰이는 transport | null,
	  producers: {
		  video: Producer | null,
		  audio: Producer | null,
		  screen: Producer | null
	  },
	  consumers: Map<Producer.id: Consumer>,
	  role: "host" | "participant",
	  joinedAt: Date
  }>,
  graceTimers: Map<userId, timerId>: 나간 사용자에대한 setTimeout,
  createdAt: Date
}>
```


순서를 먼저 말해보면
1. 프로그램 시작 시 mediasoup.Worker 생성
2. 사용자가 그룹 방 생성하기 버튼 누름
3. 서버에서 worker에게 router 생성하라고 함
4. rooms(router)에 대해서 호스트 등록
5. 사용자가 접속 요청 (호스트도 방 생성 알림 받은 후 해당 이벤트 요청 하게됨)
6. 참여하는 사용자에 대해서 produce/consume transport 관리해주는 로직 실행 
7. 사용자 이탈 시 graceTimers를 통해 몇초 후 peer 삭제하게 해주며 호스트의 경우에는 일단 timer 걸어놓고 회의방 삭제되게 하기. 명시적으로 나가기 눌렀을 경우에는 바로 삭제해주기
